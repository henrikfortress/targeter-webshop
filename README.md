# Targeter Webshop

Bestillingsportal for trykkeriprodukter. Bygget med Next.js, Better Auth, Drizzle ORM og Resend.

## Kom i gang

```bash
bun install
bun dev
```

Åpne [http://localhost:3000](http://localhost:3000) i nettleseren.
